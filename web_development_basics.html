HTML Basics

Aufbau eines HTML-Dokuments

Jede Website beginnt mit dieser Grundstruktur:

<!-- 1. Dokumenttyp: Sagt dem Browser "Das ist HTML5" --> <!DOCTYPE html> <html lang="de"> <!-- 2. Head: Unsichtbare Infos für Browser & Suchmaschinen --> <head> <meta charset="UTF-8"> <title>Meine Website</title> <link rel="stylesheet" href="style.css"> </head> <!-- 3. Body: Alles, was man auf der Seite sieht --> <body> <header>Navigation & Logo</header> <main>Hauptinhalt</main> <footer>Impressum & Links</footer> </body> </html>

HTML Basics

Die wichtigsten HTML-Tags

📝 Text-Elemente

<!-- Überschriften (h1-h6) --> <h1>Hauptüberschrift</h1> <h2>Unterüberschrift</h2> <!-- Absatz --> <p>Das ist ein Textabsatz.</p> <!-- Hervorhebungen --> <strong>Fett</strong> <em>Kursiv</em> <!-- Link --> <a href="https://...">Klick mich</a>

🖼️ Medien & Container

<!-- Bild einfügen --> <img src="foto.jpg" alt="Beschreibung"> <!-- Video einbetten --> <video src="clip.mp4" controls></video> <!-- Container (keine Bedeutung, nur Gruppierung) --> <div> Inhalt hier </div> <!-- Inline-Container --> <span>Text</span>

CSS – Cascading Style Sheets

Jetzt wird's bunt! CSS macht aus langweiligem HTML schöne Designs.

CSS Basics

CSS einbinden – 3 Wege

1. Externe Datei ⭐

Best Practice!

<!-- Im <head> --> <link rel="stylesheet" href="style.css">

2. Im <style>-Tag

Für kleine Projekte

<head> <style> body { color: red; } </style> </head>

3. Inline-Style

Nur für Ausnahmen!

<p style="color: blue;"> Blauer Text </p>

CSS Basics

Selektoren – Elemente ansprechen

Mit Selektoren sagst du CSS, welche Elemente gestylt werden sollen:

/* 1. Element-Selektor: Alle <p>-Tags */ p { color: darkgray; } /* 2. Klassen-Selektor: Alle Elemente mit class="highlight" */ .highlight { background: yellow; } /* 3. ID-Selektor: Das EINE Element mit id="logo" */ #logo { width: 100px; } /* 4. Kombiniert: Nur <a> innerhalb von <nav> */ nav a { text-decoration: none; } /* 5. Pseudo-Klasse: Beim Hover */ button:hover { transform: scale(1.1); }

CSS Basics

Die wichtigsten CSS-Eigenschaften

🎨 Farben & Text

color: #333; /* Textfarbe */ background: #f0f0f0; /* Hintergrund */ font-size: 18px; /* Schriftgröße */ font-weight: bold; /* Fett */ text-align: center; /* Zentrierung */

📦 Box-Modell

width: 300px; /* Breite */ padding: 20px; /* Innenabstand */ margin: 10px; /* Außenabstand */ border: 2px solid #000; /* Rahmen */ border-radius: 10px; /* Runde Ecken */

✨ Layout & Effekte

/* Flexbox (für Layouts!) */ display: flex; justify-content: center; align-items: center; gap: 20px; /* Schatten */ box-shadow: 0 4px 15px rgba(0,0,0,0.1); /* Animationen */ transition: all 0.3s ease; /* Transparenz */ opacity: 0.8;

JavaScript Basics

JavaScript einbinden

Externe Datei ⭐

Empfohlen!

<!-- Am Ende des <body> --> <script src="script.js"></script>

Im <script>-Tag

<script> console.log('Hallo Welt!'); </script>
💡
Tipp: Öffne die Browser-Konsole mit F12 → „Console", um console.log() Ausgaben zu sehen!

JavaScript Basics

Grundlegende Programmierkonzepte

// 1. VARIABLEN - Speichern von Werten let name = "Anna"; // Kann sich ändern const PI = 3.14159; // Bleibt konstant // 2. DATENTYPEN let zahl = 42; // Number let text = "Hallo"; // String let istWahr = true; // Boolean (true/false) let liste = ["a", "b", "c"]; // Array (Liste) let person = { // Object name: "Anna", alter: 25 }; // 3. FUNKTIONEN - Wiederverwendbarer Code function sagHallo(name) { return "Hallo, " + name + "!"; } console.log(sagHallo("Welt")); // → "Hallo, Welt!"

JavaScript Basics

Bedingungen & Schleifen

🔀 If / Else

let alter = 18; if (alter >= 18) { console.log("Volljährig! 🎉"); } else { console.log("Noch nicht 18."); }

🔄 Schleifen

// For-Schleife for (let i = 0; i < 5; i++) { console.log(i); // 0,1,2,3,4 } // Über Array iterieren const farben = ["rot", "grün", "blau"]; farben.forEach(farbe => { console.log(farbe); });

JavaScript Basics

Das DOM – Elemente manipulieren

Mit JavaScript kannst du HTML-Elemente finden, ändern und auf Aktionen reagieren:

// 1. ELEMENT FINDEN const button = document.querySelector('.mein-button'); const alleCards = document.querySelectorAll('.card'); const logo = document.getElementById('logo'); // 2. INHALT ÄNDERN button.textContent = 'Neuer Text'; button.innerHTML = '<strong>Fetter Text</strong>'; // 3. STYLING ÄNDERN button.style.backgroundColor = 'coral'; button.classList.add('aktiv'); button.classList.toggle('versteckt'); // 4. AUF EVENTS REAGIEREN button.addEventListener('click', () => { alert('Button wurde geklickt!'); });

🌐 Was macht der Browser?

  • Sendet HTTP-Anfragen an Server
  • Empfängt HTML, CSS, JS, Bilder
  • Parsed HTML → baut DOM-Baum
  • Wendet CSS an → Render Tree
  • Führt JavaScript aus
  • Zeichnet die Pixel auf dem Bildschirm

📡 DNS – Das Telefonbuch des Internets

Übersetzt lesbare Adressen in IP-Adressen:

google.com → 142.250.185.78 github.com → 140.82.121.4

Jeder Server hat eine eindeutige IP-Adresse!
